# The Iowa Action Desk

Low-tech civic routing infrastructure centered on economic solidarity and structural community advocacy. This platform is maintained by an independent private citizen operating in a personal capacity to support democratic self-determination.

## Active Project Goals
* Maintain a zero-maintenance, manual directory routing tree written strictly in Markdown.
* Optimize all user interfaces to scale vertically for flawless rendering on a 320px mobile viewport.
* Structure every single civic action alert around the professional social work principles of systems theory and the person-in-environment perspective.
* Provide an alternative perspective escape hatch on every script to preserve community self-determination.

## Mobile Design Requirements
* **Viewport Scale:** No text element or block-level link container forces horizontal scrolling on a 320px wide viewport.
* **Touch Target Safety:** Interactive link blocks must use full vertical padding blocks to ensure a minimum click target height of 48px.
* **Punctuation Audit:** Strict prohibition against the use of emojis, graphic symbols, or a single em dash.
